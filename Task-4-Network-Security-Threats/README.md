# Task 4: Common Network Security Threats

## Objective
The objective of this task is to study and understand common network security threats that affect individuals and organizations. This task focuses on identifying different types of threats, their impacts, real-world examples, and preventive measures to enhance network security awareness.

---

## Description
Network security threats are malicious activities designed to compromise the confidentiality, integrity, or availability of network resources. In this task, various network threats such as DoS, DDoS, MITM attacks, spoofing, packet sniffing, and DNS poisoning are analyzed in detail.

The task helps in understanding how attackers exploit network vulnerabilities and how organizations can defend against these threats using best security practices.

---

## Topics Covered
- Introduction to Network Security Threats  
- Types of Network Security Threats  
  - Denial of Service (DoS) and Distributed DoS (DDoS)  
  - Man-in-the-Middle (MITM) Attacks  
  - Spoofing  
  - Packet Sniffing / Eavesdropping  
  - DNS Spoofing / Poisoning  
- Impact of Network Security Threats  
- Real-World Examples  
- Mitigation and Preventive Measures  
- Best Practices for Network Security
